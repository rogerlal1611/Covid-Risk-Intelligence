# 🌍 COVID Risk Intelligence Dashboard

A real-time interactive dashboard that visualizes COVID-19 risk levels across Indian states using Leaflet.js and live API data.

---

## 🚀 Features

- 🗺️ Interactive dark-themed map (Leaflet.js)
- 🔴 Dynamic risk-based circles (Safe → High Risk)
- 🔎 Smart autocomplete search
- 📍 User geolocation tracking
- 🎯 Risk filtering system
- 📊 Live API integration

---

## 🧠 Tech Stack

- HTML5
- CSS3 (Glassmorphism UI)
- JavaScript (Vanilla)
- Leaflet.js
- REST API (Rootnet COVID data)

---

## 📡 API Used

- https://api.rootnet.in/covid19-in/stats/latest
